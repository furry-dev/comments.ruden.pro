Описание проекта

Проект предоставляет удобное API для взаимодействия с комментариями. На серверной стороне вы можете легко создавать, редактировать и удалять комментарии, а также ставить оценки. Пользователям предоставляется свобода в дизайне комментариев и полном контроле через HTTP запросы.

Дополнительный уровень безопасности

При создании ветки есть возможность передать токен приложения, зарегистрированного на account.ruden.pro. Для работы с комментариями потребуется не только наличие идентификатора ветки, который вы получаете при её создании, но и передача токена приложения через заголовок "Authentication bearer". Планируется также добавить функционал, позволяющий управлять ветками через вход в аккаунт ruden.pro на этом сайте.
